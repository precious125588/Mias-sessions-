# Mias MDX — Session Generator

Pairing server for Mias MDX WhatsApp Bot.

Sessions start with `prezzy_` prefix and last 30+ days.

### Features
- **Pair Code** — Enter your number, get a code, link in WhatsApp
- **QR Code** — Scan QR from WhatsApp Linked Devices
- **Validate** — Check if a session string is valid before using it

### Setup
```bash
npm install
npm start
```

### Deploy
Deploy to Render, Railway, or any Node.js host.
